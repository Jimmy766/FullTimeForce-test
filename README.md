FULLTIMEFORCE - TEST
===


## Install & Dependence
- nodejs
- git


## Step 1: Clone project

  ```
git clone https://github.com/Jimmy766/FullTimeForce-test.git
  ```

## Step 2: Install dependencies (backend - frontend)
- backend
```
cd backend
  ```
  
  ```
npm i
  ```
- front end
```
cd frontend
  ```
  
  ```
npm i
  ```


## Step 3: Run project (backend - frontend)
- backend
  
```
npm run start
  ```

Exposed endpoint:
```
http://localhost:4000/api/commits/
  ```
- front end

Exposed URL:
  
 ```
http://localhost:3000/
  ```
